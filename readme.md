
server_cloak
-----------------

![](https://github.com/pandorabox-io/server_cloak/workflows/luacheck/badge.svg)

# Overview

Disables `server_announce` if a given threshold of players is reached

# Settings

* **server_cloak.threshold** the announce threshold (default: 15)

# Compatibility

* Minetest >= 5.0

# Dependencies

None
